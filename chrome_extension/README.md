# 🎯 Francis Teams Assistant - Installation Ultra-Simple

## 🚀 Installation en 30 secondes

### 1. Télécharger l'extension
- Cliquez sur le bouton "Code" vert en haut de cette page
- Sélectionnez "Download ZIP"
- Décompressez le fichier sur votre bureau

### 2. Installer dans Chrome
- Ouvrez Chrome
- Allez dans `chrome://extensions/`
- Activez le "Mode développeur" (en haut à droite)
- Cliquez "Charger l'extension non empaquetée"
- Sélectionnez le dossier `chrome_extension` décompressé

### 3. C'est tout ! 🎉

## 🎤 Comment utiliser Francis

1. **Aller sur Microsoft Teams**
   - Ouvrez Teams dans votre navigateur
   - Francis se détecte automatiquement

2. **Cliquer sur le bouton Francis**
   - Un bouton doré "🎤 Francis" apparaît en haut à droite
   - Cliquez dessus pour l'activer

3. **Francis écoute et aide !**
   - Francis écoute votre réunion
   - Il propose des suggestions fiscales en temps réel
   - Les suggestions apparaissent en popup

## ✨ Fonctionnalités

- **🎯 Suggestions fiscales** : Francis analyse vos conversations et propose des conseils
- **📝 Transcription automatique** : Toutes les conversations sont transcrites
- **🔒 100% confidentiel** : Vos données restent privées
- **⚡ Temps réel** : Suggestions instantanées pendant vos réunions

## 🛠️ Dépannage

**Francis n'apparaît pas sur Teams ?**
- Vérifiez que vous êtes sur `teams.microsoft.com`
- Rafraîchissez la page Teams
- Vérifiez que l'extension est activée dans Chrome

**Francis n'écoute pas ?**
- Cliquez sur le bouton Francis pour l'activer
- Autorisez l'accès au microphone quand demandé
- Vérifiez que votre micro fonctionne

**Francis ne propose pas de suggestions ?**
- Parlez de sujets fiscaux dans votre réunion
- Francis a besoin de contexte fiscal pour proposer des suggestions
- Vérifiez votre connexion internet
